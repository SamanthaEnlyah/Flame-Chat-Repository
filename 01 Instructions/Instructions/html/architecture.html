Architecture


Architecture is Server-Client Socket-Based architecture.

Sockets are used for communication between server and client.

Client sends commands to Server, and server calls database methods.

Commands are sent from client to server and server to client.

Server communicates with the database.

Entity Framework used to create model and manipulate database records.

Server is broadcasting its IP address.

Users, their avatars, their personality traits, contacts and messages are persisted in the database.
