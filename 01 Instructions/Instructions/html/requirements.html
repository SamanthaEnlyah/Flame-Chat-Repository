INSTALL REQUIREMENTS (NOT INCLUDED IN DOWNLOAD)

  1. .NET runtime 9.0.4, 64-bit (windows desktop) - It is required for running the Flame Chat Server and Client.

    You will find installation here:

    x64: https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-9.0.6-windows-x64-installer

    x86: https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-9.0.6-windows-x86-installer

  2. SQL Server Express 2022 - Required for Flame Chat Server because Server communicates with the database

    Link: https://www.microsoft.com/en-us/sql-server/sql-server-downloads

  3. SQL Server Management Studio 21:

    Link: https://aka.ms/ssms/21/release/vs_SSMS.exe (This link will start download automatically.)

You need to install all these programs if you want to use Flame Chat.
