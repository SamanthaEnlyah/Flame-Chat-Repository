FEATURES


1. CLIENT:

List of implemented features:

[App is still under development]
  1. IP CHOOSER
  2. SIGN UP
  3. LOG IN
  4. LOG OUT
  5. VIEW PROFILE
  6. ADD PERSONALITY TRAIT
  7. ADD CONTACT
  8. SEND MESSAGE
  9. VIEW MESSAGES
Not implemented features:
  1. Change username
  2. Change password
  3. Delete contact
  4. Notify user if he/she has message from a user that is not his/her contact. You have to manually add contact to be able to see the message.
  5. Indicator for unseen messages
  6. Indicator for seen messages
  7. Indicator for online users

Client features:

Add personality trait and add contact are done in Profile.

Add contacts first to be able to send messages. You have to add both participants to be each others' contacts. If Isabela Parks talks to Caden Jacobs, Caden has to be logged in and has to add Isabela as his contact. Isabela also has to be logged in and add Caden to her contacts. Otherwise, they will not see each others' messages. (This will be fixed in further versions).

Send message and view messages are done in Chat Window.

You access profile from Chat Window. In the upper left corner there is a Profile button.

When you login, Profile button is disabled, because the app is loading data from database via server, and you cannot access profile during that time.

When loading is done, Profile button is enabled. Then you can go to Profile and add contacts from list of users (contacts are saved in database when added), and you can choose personality traits from list of personality traits in database (saving to database is needed for this).
