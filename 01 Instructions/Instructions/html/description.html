Flame Chat Application Description


Application is named Flame Chat.

This is application for chatting on local network.

There are users and they send messages to each other.

Users are either regular users or admin users. Admin is under development.

Every regular user can send a message to any other regular user.

Users have their profile which includes:
Username, password, personality traits and contacts.

When profile is open, user can add his/her personality traits, and view all other users and their avatars and personality traits. When user choses another user, perhaps by personality traits, he/she can add chosen user as contact, and then send messages to him/her and receive messages from this contact.

Application plays a short notification sound when the message is received.

Content of downloaded project

This project contains 4 folders.

  • 01 Instructions
  • 02 Data
  • 03 Application
  • 04 Code
01 Instructions

Instructions folder contains everything things you need to know about the project: what is it, list of features, requirements, use cases, how to use it, architecture, application flow, explanation of source code and tools used.

02 Data

Data contains sql for creating database schema and inserting records in created database. Name of the sql file is: "tables_and_records.sql".

You only need to create database named flamechat, and run the sql file.

It also contains data about users that are inserted in database. Usernames are in the file named "usernames.xlsx".

03 Application

Here is the compiled version of projects. Server, and two clients, Client 1 and Client 2. There are two clients because you will need to login two users, if you want to be able to test the talk between them. These two folders, Client 1 and Client 2, contain the same files.
