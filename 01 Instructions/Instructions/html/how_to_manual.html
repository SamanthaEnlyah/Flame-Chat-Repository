Manual: How to


I Data

Creating a database and inserting records


Open SQL Server management studio and create a database called flamechat.

Go to File -> Open -> File...

Select "02 Data/tables_and_records.sql".

Right click inside query text -> Execute. This will run the sql query.

SQL query will create tables and insert some data.

Data contains 11 users who have contacts and personality traits (over 650 traits are in db) and have exchanged some messages.

List of users is in "02 Data/usernames.xlsx" file, which you can use to login.


II Running


Compiled Server and Client are in folder "03 Application".

Server is in 03 Application/Server/net9.0-windows/FlameChatServer.exe

Client 1 is in 03 Application/Client 1/net9.0-windows/FlameChatClient.exe

Client 2 is in 03 Application/Client 2/net9.0-windows/FlameChatClient.exe

You have to allow all .exe. files to go over private network.

1. Server

Server must be started before clients.

Start the server.

When starting Server, IP Address Chooser window will appear. You need to select an IP Address from the list. You also need to type the SQL Server Express instance id. Server is the one to talk to the database, it needs this data to create connection string. Click OK to confirm your info.

2. Client

Important note

IP Address Chooser for client is appearing each time before you login and before you sign up.

Choose IP address before you click anything else, because, client is starting and connecting to server, and it must have an IP address.

Client has to be on the same network as Server.

Example: if server is on 192.168.40.X, then all clients have to be on 192.168.40.Y. Every client CAN be on the SAME IP ADDRESS as the Server, meaning it can have address: 192.168.40.X

Running the client

Start the client.

Click OK on Welcome dialog.

LogIn/SignUp Window with two buttons (LogIn and Sign Up) will appear.

Use cases

Not Logged In, want to Log In:

If you are not logged in, choose Log In.

Two windows will appear, IP Address Chooser and Log In window.

First select IP address.

After that login with existing user data.

Chat window will appear.

Wait for all data to be read from database, it will take about half-minute.

Not logged In, Want to Sign Up: If you are not logged in and you want to Sign up, click Sign up, and IP Address Chooser will appear together with Sign Up window. Select IP on IP Chooser and click Choose. THEN you can sign up. Fill in the username and password and choose picture. Click SIgn up and wait for a minute. After Signing up, you will be Logged in and Chat Window will appear.

Logged in: If you are already logged in, IP Address Chooser also appears at the same time as LogIn/Signup window. Select IP Address that is on the same network as Server. App will read from session/session.json file login data, and it will login and show Chat Window.

Logged In, Log in as different user: If you are logged in and you want to Log in as different user, click Log Out and then click Log In.

Logged In: If you logged in, Chat Window will appear.

Go to Profile to see data about user. Profile button is in upper left corner of Chat Window, and it is disabled at first. This is because the app is loading data from database via server, and you cannot access profile during that time

When loading is done, Profile button is enabled. Then you can go to Profile and add contacts from list of users (contacts are saved in database when added), and you can choose personality traits from list of personality traits in database (saving to database is needed for this).

When you are logged in, you have several options:

You can Add contacts, Add personality traits and Send and Receive messages.

Client how-to
Profile

You access profile from Chat Window. In the upper left corner there is a Profile button.

Adding personality traits and contacts

Adding personality trait and adding contacts are done in Profile.

Click Profile in Chat Window.

Profile window will open.

Adding personality traits

Every user has personality traits. User adds traits for him/her self.

User can see other users personality traits.

User can use these traits to decide who will he add as contact and talk to.

How to add traits

Go to Profile window. In upper right part of Profile window is list of your personality traits.

Click plus button to add your personality traits.

Small narrow window will appear and will contain all personality traits you can choose from.

Select a trait and click Add.

It will be added to your list.

Add as many traits as you want and then click Save button.

Close the small window with list of traits from database.

Adding contacts

You have to add contacts first to be able to send messages.

You have to add both participants to be each others' contacts.

If Chase Walters talks to Kyra Villa, Chase has to be logged in and has to add Kyra as his contact.

Kyra also has to be logged in and add Chase to her contacts.

Otherwise, they will not see each others' messages. (This will be fixed in further versions).

How to add contacts

Log In as first user (for example, Chase Walters).

Click Profile in Chat window.

In Profile window, notice list of users at the lower-right part of window.

This list contains all users in the database.

Click the user to select it.

Next to this list, there will be selected user information.

Information consists of Avatar image and personality traits.

When you click on user, this will be shown.

To add a user as contact, there is a plus button below the list of users.

Select Kyra Villa and click Add button.

When you click this button, user will be added to Contact list, both in Profile window and in Chat Window. Manual saving is not neccesary.

Kyra Villa will then be a contact of Chase Walters.

Now Kyra Villa has to add Chase as her contact.


Leave this user (Chase) logged in.

Log In as another user (Kyra Villa), who will talk to the first user (Chase).

Follow previous steps to add a contact. (Chose Chase from the list of users and click Add (Plus) button.)

After both users added each other as contacts, they can communicate.

Sending messages

Send message and view messages are done in Chat Window.

Chat window has list of messages and text field to type message in.

Chat window has info about logged user: an image (avatar) is to the left, and, between Profile button and image, is username.

Chat window has list of contacts in the lower left corner.

When contact is selected, user info on window appear. This contact info is at the right of the window. There is username to the right of the message list and above that is contact username.

So, image to the left is logged user, and image to the right is his/her contact.


How to send messages

You have two Chat windows open, one is form Kyra and the other is Chase.

Click on Chat window of user that will send a message (For example Chase).

Now, Chase should select contact to talk to. Select Kyra Vila from contacts.

Chase can type his message in message box.

When typing is over, Chase will click big purple button in a shape of arrow (or triangle)

Second later, Kyra Villa will receive this message. Sound is played when message is received.


That is all.
